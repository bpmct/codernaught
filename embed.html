1. Web Component (drop into any site)

Easiest way to put the walking bot on a page — works in plain HTML, React, Vue, Astro, etc.

<script type="module"
  src="https://bpmct.github.io/codernaught/codernaught-embed.js"></script>

<codernaught-bot walk auto-rotate
  style="width:400px;height:400px"></codernaught-bot>

Attributes: walk, speed="1", auto-rotate, no-controls, src="...glb", background.

2. iframe (zero setup)

<iframe src="https://bpmct.github.io/codernaught/?embed=1"
  width="400" height="400" frameborder="0"
  style="border:0;border-radius:12px"></iframe>

3. The GLB (canonical portable asset)

The rigged, walk-animated codernaught.glb drops into <model-viewer>, Spline, Rive (3D), Blender, Unity, or any Three.js scene. Hand this file to Claude/Claude Code and ask it to embed the bot — it's a standard glTF with a Walk animation clip.

<script type="module"
  src="https://unpkg.com/@google/model-viewer"></script>

<model-viewer src="https://bpmct.github.io/codernaught/codernaught.glb"
  autoplay animation-name="Walk" camera-controls
  style="width:400px;height:400px"></model-viewer>
