1. Web Component — drop into any site

Self-contained (bundles its own Three.js). Works in plain HTML, React, Vue, Astro, etc.

<script type="module"
  src="https://bpmct.github.io/codernaught/codernaught-embed.js"></script>

<codernaught-bot walk style="width:360px;height:360px"></codernaught-bot>

Attributes: walk, spin="35" (deg/sec), no-controls, bg, src.

2. iframe — zero setup

<iframe src="https://bpmct.github.io/codernaught/?embed=1"
  width="360" height="360" frameborder="0"
  style="border:0;border-radius:12px"></iframe>

3. GLB — the canonical portable asset

The rigged, walk-animated codernaught.glb drops into Spline, Rive (3D), Blender, Unity, <model-viewer>, or any Three.js / AI-generated site. Hand it to Claude / Claude Code to add the bot anywhere.

<script type="module" src="https://unpkg.com/@google/model-viewer"></script>
<model-viewer src="https://bpmct.github.io/codernaught/codernaught.glb"
  autoplay animation-name="Walk" camera-controls
  style="width:360px;height:360px"></model-viewer>
